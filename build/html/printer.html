Printer Installation¶

Keller Williams Bluegrass uses special printers agents can use from any computer they choose. These printer are enterprise grade with a special setup for cost accounting. In order to print you will need a pin code provided by the market center and this pin code will need to be used whenever you would like to print.

Windows 10¶
